<h1 align="center">CutterKit. Hard task? Easy solution!</h1>
An open ecosystem of software to simplify video creation with public source code is now a reality!<br>
All the programs posted here share a common goal: helping you edit various parts of a live broadcast for platforms like YouTube/TikTok.
