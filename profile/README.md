<h1 align="center">CutterKit. Hard task? Easy solution!</h1>
An open ecosystem of software to simplify video creation with public source code is now a reality!<br>
All the programs posted here share a common goal: helping you edit various parts of a live broadcast for platforms like YouTube/TikTok.

## Current projects

| Project | Description | Status | Dev date | Release date |
| - | - | - | - | - |
| FragmaLive | Downloading specific parts of a live broadcast from Twitch, with support for multiple timestamps. | 🔵 In dev | - | - |
| AutoREC | Automatically recording a live broadcast immediately after it starts. Desktop notifications can be configured. | 🔵 In dev | - | - |
| Vertical | Automatically generates videos for YouTube Shorts/TikTok, prioritizing longer videos as a source. | 🔵 In dev | - | - |
| NoQuiet | Removing parts of the video that do not contain speech/music/any other sounds. | 🔵 In dev | - | - |
| SubGen | Subtitle generation using AI. | 🔵 In dev | - | - |
| MetaGen | Generate video title, description, and tags/hashtags. | 🔵 In dev | - | - |
| Thumby | Create a video preview using a custom template. | 🔵 In dev | - | - |
| CopyCheck | Checking the finished video for copyright issues. | 🔵 In dev | - | - |
| ChatSave | Logging live stream chat on Twitch. | 🔵 In dev | - | - |
